# Example: Creative Flow

## User request

"Come up with a unique repo name for this gentle curiosity skill."

## Poor agent mode

- Produces generic names.
- Optimizes only for literal keywords.
- Does not capture the exploration-current concept.

## FlowCurrent setup

```text
Objective: Name the repo.
Curiosity object: What name captures gentle curiosity as internal task flow rather than soft questioning?
Accepted reachability: A distinctive name can emerge by combining source concepts: gentle curiosity, flow, current, exploration, reachability.
Feedback: uniqueness, memorability, relevance, install-friendly slug.
```

## Exploration

Directions:

- compass: guidance, but too conversational;
- current: movement, flow, surfing exploration;
- forge: building, but too forceful;
- stream: gentle but possibly generic;
- flowcurrent: direct, distinctive, agent-operational.

## Result

Recommended repo name:

```text
gentle-curiosity-flowcurrent-skills
```

Rationale: includes the required gentle curiosity phrase, while "flowcurrent" captures the internal movement of exploration toward flow.
