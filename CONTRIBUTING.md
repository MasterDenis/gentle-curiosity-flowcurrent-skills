# Contributing

Contributions should preserve the core purpose of this skill: improving the agent's internal mode of task execution through gentle curiosity, accepted reachability, and feedback-guided immersion.

## Good contributions

- Clearer operating protocols.
- Better task-domain adaptations.
- Safety improvements.
- More precise examples and eval cases.
- Better compatibility with Hermes Agent or other Agent Skills-compatible runtimes.

## Avoid

- Turning the skill into generic motivational language.
- Making claims that AI agents literally experience human flow states.
- Adding hidden persuasion, dependency-building, or manipulative tactics.
- Adding executable scripts that send data to external services.
- Making the agent ask unnecessary user questions instead of working with available context.

## Style

Use crisp operational instructions. The agent should know what to do, when to do it, and how to verify the result.
